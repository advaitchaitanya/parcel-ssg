# Parcel SSG
> Parcel template for building static sites

## Develop on a local server with hot reloading
`npm start`

## Lint CSS and JS with [XO](https://github.com/xojs/xo)
`npm run lint`

## Build for prodution
`npm run build`

## Deploy on [GitHub Pages](https://pages.github.com/)
`npm run deploy`
